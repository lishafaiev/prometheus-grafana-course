# Моніторинг з Prometheus та Grafana

Мій робочий репозиторій по курсу [robotdreams](https://robotdreams.cc/uk/course/2212-monitoring-z-prometheus-ta-grafana) (12 занять, 16 червня – 23 липня). Тримаю тут конспекти, конфіги, ДЗ і все, що напрацював по ходу — щоб не губилось і щоб бачити, як росте мій сетап.

## Мій процес

- на кожне заняття заводжу гілку `lesson-NN`;
- конспект і конфіги пишу одразу у файли під час уроку;
- наприкінці роблю коміт;
- для здачі беру готові файли з папки заняття і вантажу на платформу robotdreams.

## Структура

```text
lessons/      — по папці на заняття (конспект + конфіги + ДЗ)
dashboards/   — експортовані дашборди Grafana
notes/        — шпаргалки (PromQL тощо)
configs/      — спільні конфіги
```
