## Робочий процес

1. Нова гілка на заняття: `git checkout -b lesson-02`
2. Конспект і конфіги пишу одразу у файли під час уроку.
3. Коміт наприкінці: `git commit -m "lesson 02: ..."`
4. Файли для здачі ДЗ беру готовими з папки заняття.

## Структура

```
lessons/      — по папці на заняття (конспект + конфіги + ДЗ)
dashboards/   — експортовані JSON-дашборди Grafana
notes/        — наскрізні нотатки, шпаргалки
configs/      — спільні конфіги
```
